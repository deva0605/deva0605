# Production Monitoring & Alerting Stack

## 📖 What This Project Does

This project implements a production-style monitoring and alerting system for Linux servers using Prometheus and Grafana. It continuously collects system metrics such as CPU, memory, disk, and network usage, visualizes them through interactive dashboards, and automatically sends email notifications whenever configured alert conditions are met. The project demonstrates the complete monitoring workflow used in modern DevOps environments.

**Highlights**

- Prometheus
- Grafana
- Node Exporter
- Alertmanager
- Email Notifications
- PromQL

🔗 Repository:
https://github.com/deva0605/Email-Alerts-with-Prometheus-Grafana
